## Troubleshooting

- If the loader does not open, confirm that Windows can run the downloaded file and that the folder is not restricted by permissions.
- If downloads hang, verify your network connection and try again from the latest build page.
- If the client files are not detected, make sure the loader is pointed at the correct local folder.
- If an older version appears to stay in place, clear old cache or temporary files before launching again.
- If scripts or presets fail to load properly, inspect the local config files and check that they match the current client version.
- If you are moving up from an older release, restart the loader after the download finishes so it can refresh its state.
